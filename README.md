# Everest in Crisis – Climate Change Awareness

A one‑page website presenting 6 alarming facts about how Mount Everest is being transformed by climate change, pollution, and human impact.  
Built with pure HTML/CSS (no JavaScript) as a group project for CSC 317 at San Francisco State University.

## 📱 Responsive Design
- **Desktop (>600px):** horizontal navbar, side‑by‑side content, large fonts  
- **Mobile (<600px):** hamburger menu, stacked layout, compact spacing  

The two designs are implemented using a single CSS media query – no separate files.

## ✨ Features (Pure HTML/CSS)
- Sticky navigation bar  
- Hamburger menu (checkbox hack, no JavaScript)  
- Flexbox layout throughout  
- Google Fonts (Playfair Display & DM Sans)  
- Footer with project credits  
- 9+ pseudo‑elements (`::after`, `::before`, `::first-letter`, …)  
- 11+ pseudo‑classes (`:hover`, `:first-child`, `:last-child`, `:focus`, `:active`, …)
